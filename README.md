Description du projet
Ce projet est l'intégration d'une maquette Figma représentan une boutique de sneakers. L'objectif principal était de traduire fidèlement le design en code.

Le site est composé de deux pages :
index : La page d'accueil avec la navigation, le hero header, et la liste des produits avec l'historique des prix.
product : Une page de détail simulant l'action de clic sur le bouton BUY, affichant le produit en grand et un sélecteur de pointures

Technologies utilisées
HTML Pour structurer sémantiquement le contenu (utilisation des balises header, main, section, article).
CSS : Pour la mise en page et le stylisme, écrit dans un fichier séparé style.c ssselon le principe DRY.
